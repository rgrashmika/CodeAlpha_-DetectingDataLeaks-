A Java-based Data Leak Prevention (DLP) system implementing AES-256 encryption and SQL injection protection for secure cloud data storage
# 🛡️ Cloud-Vault: Data Leak Prevention System

### 🚀 Overview
A high-security Java Web Application designed to prevent unauthorized data access and leaks in cloud environments.

### 🛡️ Core Security Features
* **AES-256 Encryption:** All sensitive data is encrypted using advanced cryptographic standards before database persistence.
* **SQL Injection Shield:** Implements regex-based input validation to block malicious payloads.
* **Capability Tokens:** Uses a tokenized approach for session management and authentication.

### 🛠️ Tech Stack
* **Backend:** Java Servlets, JDBC
* **Frontend:** HTML5, CSS3 (Custom Dark Vault Theme)
* **Database:** Oracle SQL 19c/21c
* **Server:** Apache Tomcat 9
